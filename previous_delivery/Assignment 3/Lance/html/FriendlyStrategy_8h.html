Detailed Description

Header file for the FriendlyStrategy class.

Implement character (friendly or unfriendly) actions as strategy pattern Each turn, a character can proceed with:

  • move
  • attack
  • free action

Must implement these ConcreteStrategies: 1) a HumanPlayerStrategy that lets the user decide where to move, who to attack, and what free actions to take; 2) an AggressorStrategy that make the character automatically move towards and attack the player character; 3) a FriendlyStrategy that makes the character automatically move towards the character, but not attack unless attacked, in which case it adopts the AggressorStrategy.klA
